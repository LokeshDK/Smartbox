 Pre-Requisute:
 	1. Cypress should be installed in the machine. Else guide to install cypress follow the steps from https://docs.cypress.io/guides/getting-started/installing-cypress.html#npm-install
 	2. Visual Studio code should be installed in the machine. Else follow the steps https://www.toolsqa.com/blogs/install-visual-studio-code/
 	3. Git should be installed in the machine.

Steps to Execute:
	1. Clone the repository https://github.com/LokeshDK/Smartbox.
	2. Open Visual Studio Code. 
	3. Click on 'Open Workspace' under file and navigate to the folder where repository has been cloned.
	4. Select file 'smartbox.code-workspace' and the whole project will be imported into Visual Studio.
	5. Open the terminal from Terminal menu.
	6. In Terminal type './node_modules/.bin/cypress open'
	7. A Cypress window will be opened with smartbox_addGift.feature file as a link.
	8. Click on the link to execute the script.
	9. Once the user clicks on the link, Chrome browser will be launched and the script will start executing.
